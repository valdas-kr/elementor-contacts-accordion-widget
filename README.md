# Elementor Contacts Accordion Widget

This is a contacts accordion widget for ***Wordpress Elementor*** website builder. It adds a contacts accordion section, which can be expanded to show more information.

The plugin uses various style and text settings, as well as two repeater controls to display **company** and **employee** information. Clicking a company section expands it and reveals employee card elements nested within.

*Company section* shows title, description, location, schedule and social media link. *Employee section* displays name, work position, spoken languages, phone and email contacts.

The plugin was tested and developed using *Hello Elementor* theme.

# Installation

To use this widget in Elementor, download the project folder and paste it into *wp-content/plugins* directory. Then activate the plugin in Wordpress.
