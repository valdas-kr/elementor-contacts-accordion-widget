# Elementor contacts accordion Widget

This is a contacts accordion widget for ***Wordpress Elementor*** website builder. It adds a contacts accordion section, which can be expanded to show more information.

The plugin uses various style and text settings and two repeater controls which allow multiple expandable sections, that, once expanded, show select number of content cards.

# Installation

To use this widget in Elementor, first download the project folder and paste it into *wp-content/plugins* directory. Then simply activate the plugin in Wordpress.
